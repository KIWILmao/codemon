Task: Build a simple RESTful API using Node.js and Express.js

Description: You have been tasked with building a simple RESTful API that allows users to retrieve and update information about products in a database. The API should be built using Node.js and the Express.js framework.

Requirements: Create a MongoDB database with a collection of products. Each product should have a name, description, and price. Build an API using Express.js that allows users to perform the following actions: Retrieve a single product by ID Retrieve all products Update the price of a product by ID Implement error handling for common errors, such as invalid input or database errors Write unit tests to ensure that the API functions as expected
